# MJTSolutions LLC – Simulated Organization Overview

**MJTSolutions** is a fictional Managed Service Provider (MSP) created for the purpose of demonstrating an ISO 27001 Information Security Management System (ISMS) implementation and alignment with the NIST Risk Management Framework (RMF).

This organization serves as the subject environment for a GRC project.

## Business Profile

Based in Athens, Ohio, MJTSolutions operates as a technology services company that manages IT infrastructure and sensitive data on behalf of internal stakeholders and external clients. The simulated company environment includes the handling of:

- Personally Identifiable Information (PII)  
- Financial records and transactional data  
- Internal business operations data  
- Client systems and managed networks  

The organization provides managed IT services such as system administration, helpdesk support, data management, and network security for small to medium-sized businesses.

## Data Environment

As part of normal operations, MJTSolutions is assumed to process and store sensitive information including:

- Customer account details  
- Employee records  
- Financial documents and billing information  
- System logs and operational data  

Because of this, the organization is subject to strict security, privacy, and regulatory considerations, making it an ideal model for implementing structured security frameworks.

## Compliance and Security Objectives

For the purposes of this project, MJTSolutions is treated as an organization with the following goals:

- Establish an ISO 27001–aligned Information Security Management System  
- Implement risk-based security controls using the NIST RMF methodology  
- Protect confidentiality, integrity, and availability of sensitive data  
- Define formal policies, procedures, and technical safeguards  
- Demonstrate governance and continuous security improvement  

## Project Scope Context

This simulated environment is used to develop and document:

- Risk assessments  
- Security policies and procedures  
- Control mappings  
- System security plans  
- Evidence of compliance activities  

All documentation, architecture, and security controls produced in this project are designed around the fictional operations of MJTSolutions.

---

**Note:** MJTSolutions is not a real company. It exists solely as a model organization for academic and professional portfolio demonstration purposes.
