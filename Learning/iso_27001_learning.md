# Notes from ISO 27001 Annex A

## Overview:
This Annex is a table of InfoSec controls that a company can implement based on their needs.
### 1. Organizational controls
- These controls are all directive controls
- Policies and procedures for various things like IRP's, cloud control matrices, access controls, etc.
### 2. People controls
- Mostly directive controls as well but more individual based.
- Things like screenings, discipline, training, etc.
### 3. Physical controls 
- Covers most all categories of physical controls.
- Securing a room, building or device storage media.
- Protecting against unauthorized entry, disposal, power outages, etc.
### 4. Technical controls
- Covering all categories of technical/logical controls.
- Endpoint security, AAA, data protection, logging, etc.
